# Global Toy Buyer Notes

A public category and channel planning workbook for wholesale toy buyers. It focuses on product-channel fit, merchandising value, MOQ, carton data, packaging and market-specific evidence.

Published site: https://kxxxyyy.github.io/global-toy-buyer-notes/

Published by TopToyFactory, the B2B sourcing brand operated by Shantou CPS Toys Co., Ltd.
